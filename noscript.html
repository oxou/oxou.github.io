List Posts

For listing available content on the page through a similar method, review the https://oxou.github.io/data/posts.json file.

Linux users can use curl -s https://oxou.github.io/data/posts.json | jq .[].id -M to get the list of post IDs in their terminal.

Windows users on the other hand must copy the output from their web browser and view it at jsonviewer.stack.hu.

Browse Posts

If you want to see a particular post in plain-text markdown format, you can do so by changing the URL from:
    https://oxou.github.io/?post=about
to:
    https://oxou.github.io/data/post/about/index.md

Note: some browsers may download the file instead of displaying it, add the prefix view-source: to the URL to see the content.
